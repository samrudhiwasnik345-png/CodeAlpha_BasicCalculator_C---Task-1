# 🧮 CodeAlpha Calculator Program (C)

## 📌 Project Overview

This project is a **basic calculator program** developed in C language as part of the CodeAlpha Internship.
It performs all fundamental arithmetic operations using a **switch-case statement**.

---

## 🎯 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* ⚠️ Handles division by zero
* 💡 Simple and easy-to-understand logic

---

## 💻 Technologies Used

* C Programming Language
* GCC Compiler / Code::Blocks

---

## 🚀 How to Run

1. Download or clone the repository
2. Open the project in Code::Blocks or any C compiler
3. Compile the program
4. Run the executable
5. Enter two numbers to get results

---

## 📌 Sample Output

```
Enter two numbers: 12 4

--- Calculator Results ---
Addition = 16.00
Subtraction = 8.00
Multiplication = 48.00
Division = 3.00
```
